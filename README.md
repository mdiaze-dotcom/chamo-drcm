# BD_Expedientes_DRCM - Streamlit + Google Sheets (PRO)

Aplicación profesional para la actualización de expedientes en Google Sheets vía Streamlit.

## Pasos:
1. Subir archivos al repositorio.
2. En Streamlit Cloud → Secrets, pegar credenciales del Service Account bajo `gcp_service_account`.
3. Compartir la Google Sheet con el `client_email` del service account (Editor).
4. Ejecutar la app.
